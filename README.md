# 📝 React Word & Letter Counter

A simple React application that counts the number of letters and words entered by the user in real time.

This project was built as part of my React learning journey to practice **React Hooks**, **event handling**, **state management**, and **live UI updates**.

## 🚀 Features

- 🔤 Live letter counting
- 📖 Live word counting
- ⚛️ Built using React Functional Components
- 🎯 Uses the `useState` Hook
- ⌨️ Updates instantly while typing
- 📱 Simple and responsive user interface

## 🛠️ Technologies Used

- React.js
- JavaScript (ES6+)
- CSS3
- Vite

## 📚 What I Learned

While building this project, I practiced:

- React Functional Components
- The `useState` Hook
- Controlled Components
- Event Handling (`onChange`)
- Live State Updates
- String Methods:
  - `trim()`
  - `split()`
  - `length`
- Dynamic Rendering in React

## 📸 Preview

Type any text into the input field and the application will instantly display:

- Total number of letters
- Total number of words
